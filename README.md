## 🧠 About

Not just a card. A vibe.

This project is a small tribute to Michael Cera, inspired by his unique presence and awkward charm. As a big fan, I wanted to capture a bit of that personality through a simple UI component.

Blending minimalist design with smooth animations, the goal was to create an interaction that feels expressive, slightly offbeat, and memorable.

## 📦 How to run locally

```bash
git clone https://github.com/lucasslvr/michael-cera-card-animation.git
cd michael-cera-card-animation
```

Then open the `index.html` file in your browser.
